# Distribution Network Single-Phase Grounding Fault Analysis (Matlab/Simulink)

### 1. System-Level Modeling & Simulation
* **MV Grid Construction:** Built a comprehensive medium-voltage distribution network model independently using **MATLAB/Simulink**.
* **Grounding Configuration:** Precisely simulated various **neutral grounding modes** (Ungrounded, Resonant Grounding) to analyze system behavior under diverse operating conditions.

### 2. Fault Mechanism & Overvoltage Analysis
* **Waveform Characterization:** Conducted in-depth analysis of **zero-sequence current and voltage waveforms** during single-phase-to-ground (SPG) faults.
* **Insulation Impact Study:** Mastered the logic of **transient overvoltage evolution** under asymmetric fault conditions and evaluated its actual impact on the insulation tolerance of core equipment like **transformers**.

### 3. Algorithm Derivation & Criterion Verification
* **Feature Extraction:** Focused on extracting transient fault features to evaluate the sensitivity and reliability of multiple **feeder selection criteria**.
* **Engineering Rigor:** Built a logical closed-loop for troubleshooting complex engineering problems in power systems.
